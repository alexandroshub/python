About AutoSortify

AutoSortify είναι ένα Python-based automation tool που λύνει ένα από τα πιο κοινά προβλήματα:
ο χαοτικός φάκελος Downloads γεμάτος από εικόνες, έγγραφα, βίντεο και αρχεία κάθε τύπου.

Με το AutoSortify:

📂 Τα αρχεία μετακινούνται αυτόματα σε σωστούς υποφακέλους (Images, Documents, Videos, Archives, Other).

📝 Κρατάει αναλυτικό log file ώστε να ξέρετε πάντα τι οργανώθηκε.

⚡ Χρησιμοποιεί βασικές αρχές Python automation (OOP, file handling, modules, exception handling).

💡 Παρέχει μια καθαρή & επαγγελματική αρχιτεκτονική που δείχνει τις core γνώσεις του Python μέσα σε ένα ρεαλιστικό use case.


Πού υπάρχει το automation:

Η εφαρμογή αναλαμβάνει μόνη της να εντοπίσει, να κατηγοριοποιήσει και να μετακινήσει αρχεία χωρίς καμία ανθρώπινη παρέμβαση.

Η δημιουργία φακέλων, η ταξινόμηση, και η καταγραφή σε log file γίνονται πλήρως αυτόματα.
